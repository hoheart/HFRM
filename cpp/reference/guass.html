为简便起见，取t=35/60=7/12。有了上面的概念，后面的工作就迎刃而解了，由于闰年的规则是每4年多一天，每100年少一天，每400年又多一天，我们以公历公元前1年的3月0日作为原始起点，折合成Guass历就是公元前1年1月0日，则当前的日期相对于该原始起点的天数就是year/4 - year/100 + year/400 + 30*(mon-1)+7/12*mon + day + year*365，1969年11月1日（即，相当于公历的1970年1月1日）相对于Guass历公元前1年1月0日（公历公元前1年3月0日）的天数为1969/4 - 1969/100 + 1969/400 + 30*(11-1)+7/12*11 +?1 + 1969*365，这样以上两个式子一减就是结果，因为用到了减法，为了效率起见，mon可以不用减1，直接用mon，这样第一个式子就成了天数就是year/4 - year/100 + year/400 + 367*mon/12 + day + year*365，而719499就是将1969、11、1三个数带进去的结果。
